## Getting Started

### Logging In

1. Navigate to the PRM Dashboard URL provided by your administrator
2. Enter your email address and password
3. Click "Sign In"
4. (Optional) Complete two-factor authentication if enabled

### First Time Setup

After your first login:

1. **Update Your Profile**
   - Go to Settings → Profile
   - Update your name, contact information, and department
   - Upload a profile picture (optional)

2. **Configure Notifications**
   - Go to Settings → Notifications
   - Enable email and/or SMS notifications
   - Select which events you want to be notified about

3. **Familiarize with the AI Assistant**
   - The AI chat panel is always visible on the right side
   - Try asking "What can you help me with?"
   - Press `Cmd+K` (Mac) or `Ctrl+K` (Windows) to open the command bar

## Dashboard Overview

The main dashboard shows you a quick overview of your day:

### Key Metrics
- **Today's Appointments**: Number of appointments scheduled for today
- **Active Journeys**: Number of patient care journeys in progress
- **Messages Sent**: Communication activity
- **Pending Tickets**: Open support tickets requiring attention

### Sections
- **Upcoming Appointments**: Next 5 scheduled appointments
- **Recent Communications**: Latest patient messages
- **AI Suggestions**: Smart recommendations based on your activity

## Using the AI Assistant

The AI Assistant is your primary interface for all operations.

### Chat Panel (Always Visible)

Located on the right side of the screen:

1. **Type your request** in natural language
2. **Press Enter** or click Send
3. **Review the plan** - AI will show you what it will do
4. **Confirm or modify** the action
5. **See results** immediately

**Examples:**
```
"Book appointment for John Doe tomorrow at 2pm"
"Show me all patients with appointments this week"
"Send reminder to all patients with appointments tomorrow"
"Create a post-surgery journey for patient Sarah"
"Find patient with phone number 555-123-4567"
```

### Command Bar (Cmd+K)

Quick access to any function:

1. Press `Cmd+K` (Mac) or `Ctrl+K` (Windows)
2. Start typing what you want to do
3. Select from suggestions or press Enter
4. AI will execute your command

**Tips:**
- Use natural language - the AI understands context
- Be specific about patient names, dates, and times
- The AI will ask for clarification if needed
- Previous conversations are remembered in the session

### Voice Input (Optional)

If enabled:

1. Click the microphone icon
2. Speak your request clearly
3. AI will process and confirm
4. Approve or modify the action

**Via Interface:**

## Managing Appointments

### Viewing Appointments

The appointments page offers multiple views:

- **Day View**: Hourly schedule for selected date
- **Week View**: 7-day calendar with all appointments
- **Month View**: Monthly calendar overview
- **List View**: Scrollable list with filters

### Booking an Appointment

**Via AI (Recommended):**
```
"Book appointment for John Doe with Dr. Smith tomorrow at 10am"
"Schedule follow-up for patient Sarah next week, 30 minutes"
"Find available slots for cardiology next Monday"
```

**Via Interface:**
1. Go to Appointments
2. Click "New Appointment"
3. Select or search for patient
4. Choose date and time
5. Select appointment type
6. Add notes (optional)
7. Click "Book Appointment"

### Rescheduling

**Via AI:**
```
"Reschedule John Doe's appointment to next Tuesday at 2pm"
```

**Via Interface:**
1. Click on the appointment
2. Click "Reschedule"
3. Select new date/time
4. Click "Update Appointment"

### Canceling

**Via AI:**
```
"Cancel appointment for John Doe tomorrow"
```

**Via Interface:**
1. Click on the appointment
2. Click "Cancel"
3. Provide cancellation reason
4. Click "Confirm Cancellation"

### Appointment Reminders

**Automatic:**
- Patients receive automatic reminders 24 hours before
- Additional reminder sent 2 hours before

**Manual:**
```
"Send appointment reminder to John Doe"
"Remind all patients with appointments tomorrow"
```

## Managing Journeys

Journeys are structured care pathways for patients.

### Journey Types
- **Post-Surgery**: Recovery and follow-up care
- **Chronic Disease**: Ongoing disease management
- **Wellness**: Preventive care and health optimization
- **Pregnancy**: Prenatal and postnatal care
- **Custom**: Create your own journey type

### Creating a Journey

**Via AI:**
```
"Create post-surgery journey for patient John with 5 steps"
"Start wellness journey for Sarah Miller"
```

**Via Interface:**
1. Go to Journeys
2. Click "New Journey"
3. Select patient
4. Choose journey type
5. Add title and description
6. Click "Create Journey"

### Managing Journey Steps

Each journey consists of steps that guide patient care.

**Adding Steps:**
```
"Add step 'First follow-up appointment' to journey for John"
```

**Completing Steps:**
```
"Mark step 2 complete for John's journey"
```

**Via Interface:**
1. Open the journey
2. Click "Add Step"
3. Enter step details
4. Set due date
5. Click "Add Step"

### Tracking Progress

The journey page shows:
- Progress bar (percentage complete)
- Upcoming steps
- Completed steps
- Overdue steps (highlighted in red)

## Support Tickets

Manage patient support requests and internal issues.

### Creating a Ticket

**Via AI:**
```
"Create billing ticket for patient John Doe"
"Create urgent technical ticket about login issue"
```

**Via Interface:**
1. Go to Tickets
2. Click "New Ticket"
3. Select category (billing, appointment, medical, technical)
4. Set priority (low, medium, high, urgent)
5. Enter title and description
6. Assign to team member (optional)
7. Click "Create Ticket"

### Managing Tickets

**Updating Status:**
```
"Mark ticket #123 as in progress"
"Resolve ticket #456 with solution notes"
```

**Adding Comments:**
```
"Add comment to ticket #123: Contacted patient, awaiting response"
```

**Assigning:**
```
"Assign ticket #789 to user Jane Smith"
```

### Ticket Categories

- **Billing**: Payment and insurance issues
- **Appointment**: Scheduling conflicts or changes
- **Medical**: Clinical questions or concerns
- **Technical**: System or access issues
- **General**: Everything else

## Best Practices

### Efficiency Tips

1. **Use the AI Assistant First**
   - Faster than navigating menus
   - Handles complex multi-step tasks
   - Natural language interface

2. **Learn Key Commands**
   - "Show patients with appointments today"
   - "Book follow-up for [patient name]"
   - "Send reminder to patients tomorrow"

3. **Use Keyboard Shortcuts**
   - `Cmd+K`: Open command bar
   - `Esc`: Close modals/dialogs
   - `/`: Focus search

4. **Leverage Templates**
   - Create templates for common messages
   - Saves time on repetitive communications

### Data Entry Tips

1. **Be Consistent**
   - Use standard date formats
   - Format phone numbers consistently
   - Use proper capitalization

2. **Complete All Fields**
   - More data = better AI suggestions
   - Easier to search and filter
   - Better reporting

3. **Use Descriptive Notes**
   - Add context to appointments
   - Document important details
   - Help future reference

### Communication Best Practices

1. **Choose the Right Channel**
   - WhatsApp: Quick updates, reminders
   - SMS: Important time-sensitive messages
   - Email: Detailed information, documents

2. **Keep Messages Clear**
   - Use simple language
   - Include specific dates/times
   - Provide contact information

3. **Review Before Sending**
   - Double-check patient names
   - Verify dates and times
   - Confirm message content

### Security Best Practices

1. **Protect Your Login**
   - Use a strong password
   - Never share your credentials
   - Log out when finished

2. **Respect Patient Privacy**
   - Only access necessary information
   - Don't discuss patients in public
   - Follow HIPAA guidelines

3. **Session Management**
   - System logs out after 30 minutes of inactivity
   - Save your work regularly
   - Extend session if actively working

## Troubleshooting

### Common Issues

**Can't log in:**
- Verify email and password
- Check Caps Lock
- Try password reset
- Contact your administrator

**AI Assistant not responding:**
- Check internet connection
- Refresh the page
- Clear browser cache
- Try different browser

**Session timeout:**
- System logs out after 30 minutes of inactivity
- Click to extend when warned
- Save work frequently

**Appointments not showing:**
- Check date range
- Verify filters
- Check permissions
- Refresh the page
