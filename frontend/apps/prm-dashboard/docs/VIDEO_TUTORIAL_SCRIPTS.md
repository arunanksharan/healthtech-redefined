## Script Format Guide

Each script follows this format:

**Duration:** Estimated length
**Difficulty:** Beginner/Intermediate/Advanced
**Prerequisites:** Required knowledge
**Learning Objectives:** What viewers will learn

**Script Sections:**
- **Narration:** What to say (in regular text)
- **[ACTION]:** What to show on screen (in brackets)
- **[VISUAL]:** Graphics, highlights, or effects (in brackets)
- **[PAUSE]:** Brief pause for emphasis (in brackets)

## Tutorial 1: Getting Started

**Duration:** 5-6 minutes
**Difficulty:** Beginner
**Prerequisites:** None
**Target Audience:** All new users

- Log into the PRM Dashboard
- Navigate the main interface
- Understand the dashboard overview
- Know where to find help

## Tutorial 4: Using the AI Assistant

**Duration:** 12-15 minutes
**Difficulty:** Intermediate
**Prerequisites:** Tutorials 1-3
**Target Audience:** All users

### Learning Objectives
- Understand what the AI Assistant can do
- Learn effective command syntax
- Use AI for patient and appointment tasks
- Handle multi-step workflows
- Use context effectively

### Script

**[INTRO SCREEN: "Using the AI Assistant"]**

**[NARRATION]**
Welcome to the most exciting tutorial yet - mastering the AI Assistant! This powerful feature can dramatically increase your productivity by handling complex tasks through simple conversational commands.

**[VISUAL: Highlight AI chat panel]**

**[NARRATION]**

**[ACTION: Click in chat input field]**

**[NARRATION]**
Let's start with a simple question.

**[ACTION: Type "What can you help me with?"]**

**[ACTION: Press Enter]**

**[VISUAL: AI responds with capabilities list]**

**[NARRATION]**
The AI explains its capabilities: patient management, appointments, journeys, communications, tickets, and more.

**[PAUSE: 2 seconds for reading]**

**[NARRATION]**
Let's try something practical. We'll create a patient using just the AI.

**[ACTION: Type "Create patient Michael Johnson, DOB 1985-03-15, phone 555-123-4567"]**

**[ACTION: Press Enter]**

**[VISUAL: AI processes request]**

**[NARRATION]**
Notice how I included all the essential information in one natural sentence.

**[VISUAL: AI shows confirmation request]**

**[NARRATION]**

**[ACTION: Click "Confirm" or type "yes"]**

**[NARRATION]**
And just like that, the patient is created! No form to fill out, no clicking through menus.

**[ACTION: Type "Book appointment for Michael Johnson tomorrow at 2pm"]**

**[ACTION: Press Enter]**

**[VISUAL: AI displays filtered patient list]**

**[NARRATION]**

**[NARRATION]**

## Production Notes

### General Guidelines

**Video Production:**
- **Resolution:** 1920x1080 (Full HD minimum)
- **Frame rate:** 30 fps or 60 fps
- **Format:** MP4 (H.264 codec)
- **Audio:** Clear narration, remove background noise
- **Mouse cursor:** Enabled and highlighted for visibility

**Screen Recording:**
- Use demo data (no real patient information)
- Slow mouse movements for clarity
- Pause between major steps
- Use zoom/highlight for small UI elements

**Editing:**
- Add chapter markers for easy navigation
- Include captions/subtitles
- Use smooth transitions
- Add background music (low volume, non-distracting)

**Branding:**
- Include company logo in intro/outro
- Use brand colors for highlights and callouts
- Add contact information in outro

### Visual Effects

**Highlights:**
- Yellow circle for clickable elements
- Blue box for important areas
- Red for warnings/critical actions

**Text Overlays:**
- Use for step numbers
- Keyboard shortcuts
- Important notes
- Best practices

**Transitions:**
- Fade for scene changes
- Wipe for menu transitions
- Zoom for detail views

### Accessibility

- Add closed captions
- Provide transcripts
- Use high contrast for highlights
- Ensure readable font sizes

### Distribution

**Platforms:**
- Company learning management system
- YouTube (unlisted or private)
- Embedded in help center
- Email onboarding sequences

**Formats:**
- Full tutorials (8-15 min)
- Quick tips (1-2 min)
- FAQ videos (30-60 sec)
